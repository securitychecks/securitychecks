# Contributing

## Development

Requirements:
- Node.js >= 18
- pnpm (this repo uses `pnpm@9.15.0`)

Commands:

```bash
pnpm install
pnpm lint
pnpm typecheck
pnpm test
pnpm build
```

## Pull Requests

- Keep changes focused and well-scoped.
- Add or update tests for behavior changes.
- Avoid committing generated artifacts (`dist/`, `.securitychecks/`).
